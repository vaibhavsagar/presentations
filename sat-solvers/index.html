SAT Solvers?

Boolean Satisfiability Problem

  • Problems of the form (x ∨ y ∨ z) ∧ (x ∨ ¬y) ∧ (¬z) (CNF)
  • NP-complete (!)
  • Can express any other NP-complete problem!

NP-complete problems

  • Knapsack problem
  • Travelling salesman problem
  • Subset sum problem
  • Graph coloring problem
  • Sudoku (!)

Sudoku

Rules

  • one number per cell
  • each number once per row
  • each number once per column
  • each number once per 3x3 sub-grid
  • must respect existing values

Insight

  • 9 boolean variables for each cell!
  • xr, c, 1, xr, c, 2,…,xr, c, 9

Each cell has at least one value

for r in range(1,10):
    for c in range(1, 10):
    add_clause(or(x[r][c][n] for n in range(1,10)))

Each row has all values

for r in range(1,10):
    for n in range(1,10):
        add_clause(or(x[r][c][n] for c in range(1,10)))

Each cell has at most one value

for r in range(1,10):
    for c in range(1,10):
        for i in range(1,10):
            for j in range(i,10):
                add_clause(or([not(x[r][c][i]), not(x[r][c][j])]))

Davis-Putnam-Logemann-Loveland

  • Backtracking search
  • Unit propagation
  • Pure literal elimination

Downsides

  • Potentially finds the same conflict multiple times
  • Backtracks one level at a time (chronologically)
  • No memory of past conflicts

Can we do better?
