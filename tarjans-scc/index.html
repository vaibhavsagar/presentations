Topological sorting

Definition

  • A linear ordering of its vertices such that for every directed edge uv from vertex u to vertex v, u comes before v in the ordering.

Why Tarjan's SCC is awesome

  • It computes the SCCs and a reverse topological sort at the same time!

2SAT

Definition

  • Can a collection of boolean variables with constraints on pairs of variables be assigned values satisfying all the constraints?

It's a graph problem

  • (u || v) == (not u => v) || (not v => u)
  • find the SCCs
  • if a SCC contains u and not u, it's unsatisfiable
  • otherwise, topologically sort the results and use this to assign T and F to the variables
